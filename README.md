
📖 Quote Generator with History

📌 Project Overview

Quote Generator with History is a Full Stack web application that fetches random inspirational quotes from an external API, displays them on the webpage, and stores every generated quote in a MongoDB database. Users can also view the complete history of generated quotes.

🚀 Features

Generate random quotes from an external API.

Display the quote and author.

Save every generated quote in MongoDB.

View the complete quote history.

Dynamic frontend and backend integration.

▶️ How to Run

1. Install dependencies:

npm install


2. Start MongoDB.


3. Run the server:

node server.js


4. Open your browser and visit:

http://localhost:5000
